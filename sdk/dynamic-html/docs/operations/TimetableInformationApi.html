This is the API

getTimeTable

URL

https://data.dublinked.ie/cgi-bin/rtpi//timetableinformation

HTTP Method

GET

Response Type

TimetableList

Parameters

  • Query: type String

    value: week; Time table information is retrieved for the whole week

  • Query: stopid String

    The bus stop id, if not provided all bus stops are returned

  • Query: routeid String

    The route identifier

  • Query: format String

    If not provided JSON format will be used by default
