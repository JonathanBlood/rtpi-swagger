This is the API

getRealTimeBusInformation

URL

https://data.dublinked.ie/cgi-bin/rtpi//realtimebusinformation

HTTP Method

GET

Response Type

RealTimeInformationList

Parameters

  • Query: stopid String

    The bus stop id

  • Query: format String

    If not provided JSON format will be used by default

  • Query: operator String

    If provided retrieves routes for the given operator

  • Query: routeid String

    The route identifier

  • Query: maxresults String

    The maximum number of results, if not provided system will limit the number of results depending on server configuration
