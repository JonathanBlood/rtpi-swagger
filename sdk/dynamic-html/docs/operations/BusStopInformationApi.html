This is the API

getBusStops

URL

https://data.dublinked.ie/cgi-bin/rtpi//busstopinformation

HTTP Method

GET

Response Type

BusStopList

Parameters

  • Query: format String

    If not provided JSON format will be used by default

  • Query: operator String

    If provided retrieves routes for the given operator

  • Query: stopid String

    The bus stop id, if not provided all bus stops are returned

  • Query: stopname String

    The stop name, only matching stops are returned
